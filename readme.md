
# food file
登録したいお店を簡単に登録できるアプリです
# Description
URLから必要な情報を取得し、簡単に登録できます。

# Requirements
- PHP 7.2.33
- Laravel Framework 5.8.38
#Note
登録画面でジャンルを記入するとジャンルごとに検索できます。
住所を入力してMAPナンバーを取得して登録するとMAPにピンが立ちます。
